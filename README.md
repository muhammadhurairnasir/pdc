## Repository Structure

The repository is organized as follows:

```
Main Branch
  |
  ├── Assignment Codes
  │     └── Contains all Python (.py) or C (.c) scripts for assignments.
  │
  ├── Assignment Docx Files
  │     └── Contains all DOCX word files or reports of assignments.
  │
  ├── Midterm
  │     └── Contains codes and reports related to midterm assignments.
  │
  └── Final
        └── Contains codes and reports related to final assignments.
```

## Details

* **Assignment Codes:**

  * Each script corresponds to a particular assignment.
  * Scripts are named appropriately for easy identification.

* **Assignment Docx Files:**

  * Contains the detailed reports in DOCX format.

* **Midterm & Final Folders:**

  * Include both codes and reports specific to the respective exams.

## Notes

* Ensure you maintain the folder structure when adding new assignments.
* Each new assignment should have both the code and the corresponding report.
* This structure allows easy navigation and retrieval of assignments and reports.
